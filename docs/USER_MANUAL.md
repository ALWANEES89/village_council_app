# USER_MANUAL — دليل المستخدم (العضو)

> دليل استخدام التطبيق للعضو العادي في المجلس. للمشرفين راجع `ADMIN_MANUAL.md`.

## 1. الدخول
1. افتح التطبيق → شاشة تسجيل الدخول.
2. أدخل رقم الهاتف وكلمة المرور → دخول.
3. عند أول انضمام، قد تحتاج التسجيل ثم طلب الانضمام لمجلس.

## 2. الانضمام إلى مجلس
- عبر **رابط/QR دعوة** (`communityos://join`) أو من داخل التطبيق: أدخل بيانات الطلب.
- يُرسل الطلب إلى إدارة المجلس للمراجعة (حالة: قيد المراجعة).
- تصلك إشعارات عند القبول/الرفض.
- إن كنت عضواً في أكثر من مجلس، تظهر شاشة **اختيار المجلس**.

## 3. الشاشة الرئيسية للعضو
- ملخّص عضويتك، المدفوعات المستحقّة، والإجراءات المتاحة.
- جرس الإشعارات أعلى الشاشة (عدد غير المقروء).

## 4. رفع إيصال دفع
1. من الرئيسية اختر الدفعة → **رفع إيصال**.
2. اختر صورة/PDF للإيصال (حد أقصى 10MB).
3. أدخل المبلغ المُعلن (إن طُلب) → إرسال.
4. تصبح الحالة "قيد المراجعة"، ويصلك إشعار عند الاعتماد/الرفض.

## 5. سجل الإيصالات والمعاملات
- **سجل الإيصالات:** كل إيصالاتك وحالتها.
- **الخط الزمني للمعاملة:** تفاصيل مراجعة إيصال معيّن (مُرسل → مُراجَع → معتمد/مرفوض).

## 6. حجز القاعة/المجلس
1. من القائمة اختر **الحجوزات** → طلب حجز جديد.
2. حدّد التفاصيل → إرسال (حالة: قيد المراجعة).
3. يصلك إشعار عند الاعتماد أو الرفض.

## 7. الملف الشخصي
- تعديل بياناتك الشخصية وصورة الملف (صور فقط، حد 5MB).
- بعض الحقول الحسّاسة يديرها المشرف فقط.

## 8. الإشعارات
- إشعارات فورية (Push) عند: قبول/رفض إيصال، تحديث عضوية/دور، نتيجة حجز، وإعلانات المجلس.
- شاشة الإشعارات تعرض السجل الكامل مع تعليم كمقروء.

## 9. الخصوصية والأمان
- ترى بياناتك ومدفوعاتك فقط.
- لا يمكنك الوصول لبيانات مجلس آخر.
- بياناتك معزولة ضمن مجلسك.

## 10. مشاكل شائعة
| المشكلة | الحل |
|---|---|
| لا تصل إشعارات | تأكّد من تفعيل الإشعارات وصلاحية الجهاز |
| فشل رفع الإيصال | تحقّق من الحجم (≤10MB) والنوع (صورة/PDF) |
| "قيد المراجعة" طويلاً | بانتظار مراجعة إدارة المجلس |
| عضوية غير نشطة | راجع إدارة المجلس |
