# نظرة عامة على المشروع

## اسم المشروع
منصة مجلس الرحمات

## الوصف
منصة متعددة المجالس Multi-Tenant مبنية باستخدام Flutter وRiverpod وFirebase.

## المصدر الرسمي
- الكود والوثائق التقنية: GitHub
- النسخة الاحتياطية: OneDrive
- المرجع التنفيذي: مجلد `project-brain`

## الأدوار الأساسية
- system_owner
- owner
- chairman
- admin
- member
- suspended

## أهداف المشروع
- إدارة عدة مجالس من منصة واحدة.
- نظام عضويات وأدوار وصلاحيات دقيق.
- اشتراكات وحجوزات مجانية أو مدفوعة.
- إدارة الإيصالات والتقارير المالية.
- OCR وميزات ذكاء اصطناعي.
- إشعارات وسجل تدقيق متكامل.
